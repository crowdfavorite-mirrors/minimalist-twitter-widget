A minimalist Twitter widget to display users tweets.

== Description ==

Minimalists Twitter Widget displays user tweets using the REST API v1.1. With minimal styling it is a good widget to build upon to fit into a theme.

To avoid API limits there is an optional Tweet cache feature which can be set to expire after a user defined amount of hours.

== Installation ==

1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

2. Go to \”Tweet Options\” under Appearance and input your Consumer Key, Consumer Secret, Access Token & Access Token Secret.

4. Go to Widgets, drag the \“Minimalist Twitter Widget\” to the your sidebar and define the widgets Title, your twitter handle and how many tweets it should display.
